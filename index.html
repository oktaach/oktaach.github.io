Peta Pembelajaran Ahli Keamanan Cyber

1. Dasar Teknologi Informasi

-Dapat memahami cara kerja OS seperti Windows, Linux, dan lain sebagainya

-Memiliki Pengetahuan dasar tentang jaringan komputer dan protokol (TCP/IP, DNS, VPN).

2. Bahasa Pemrograman dan Scripting

- Penguasaan bahasa pemrograman seperti Python untuk otomasi dan analisis.

- Pemahaman tentang scripting dan penggunaan alat keamanan yang disediakan oleh komunitas.

3. Dasar Keamanan Informatika

- Konsep dasar keamanan (kerentanan, ancaman, serangan).

- Kriptografi dasar (enkripsi, dekripsi, tanda tangan digital).

- Pengelolaan risiko keamanan.

4. Jaringan Dan Infrastruktur

- Keamanan jaringan (firewall, IDS/IPS, segmentasi jaringan).

- Keamanan server (konfigurasi yang aman, patch management).

- Keamanan aplikasi web dan database.

5. Keamanan Aplikasi

- Keamanan aplikasi web (OWASP Top 10, pengujian penetrasi aplikasi web)..

- Keamanan basis data (SQL injection, keamanan penyimpanan data).

6. Keamanan Cloud dan Virtualisasi

- Pemahaman tentang keamanan cloud (AWS, Azure, GCP).

- Keamanan lingkungan virtual (containerization, hypervisor).

7. Serangan dan Teknik Pemecahan

- Pemahaman tentang berbagai jenis serangan (DDoS, phishing, malware).

- Studi kasus serangan terkenal dan cara meresponnya.
